Datos de Ventas de Walmart
Datos de 2010-2012

El siguiente analisis es datos de ventas de diferentes tiendas de Walmart con el objetivo de buscar las mejores ventas de las tiendas y buscar las mejores epocas para las ventas. Para tal analisis se usa la libreria D3.js con los cuales se generan los graficos del siguiente dashboard, se empleara la siguiente fuente de datos: https://www.kaggle.com/datasets/mikhail1681/walmart-sales/data
Adicionalmente si se require validar el codigo fuente pueden conseguirlo en el siguiente enlace: https://github.com/leonellam07/Grupo_3_1003_Actividad_3_Herramientas_Visualizacion
